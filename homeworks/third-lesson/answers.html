Зачем нужен тег label? Как объединить label и input?

Тег label устанавливает связь между текстом (в основном), и элементом формы (input,и т.д). Такая связь необходима, чтобы изменять значения элементов формы при нажатии курсором мыши на текст. Есть 2 способа как объединить label и input. Первый заключается в использовании идентификатора id внутри элемента формы и указании его имени в качестве атрибута for тега label. При втором способе элемент формы помещается внутрь контейнера label.

Как объединить радиокнопки в группу?

Нужно задать одинаковый атрибут name для всех кнопок

Какие условия нужно соблюсти, чтобы отправить файл на сервер?

Нужно указать тег form, его атрибуты-method и action, создать кнопку для отправки форм

Что такое HTTP? Какие существуют HTTP методы?

HTTP - это сетевой протокол,с помощью которого происходит обмен данными между двумя и более подключенными к сети устройствами. Существует множество HTTP методов, самые распространенные из них POST, GET, INPUT, DELETE

Зачем используется атрибут name?

атрибут name нужен, чтобы задать имя полю (для сервера), и чтобы можно было использовать label
